# How to Clone and Run

- git clone https://github.com/phinchik/full-stack-todoList
- cd full-stack-todoList
- npm install
- npm start both server and client

# Technologies Used

- React
- Redux/Redux-Thunk
- Node
- Express
- Rest API

# Features

- Add Todo
- Delete Todo
- Update Todo
- Cancel Update mode
- Toggle Todo
- Delete All Toggled Todo
- Number of Todo
